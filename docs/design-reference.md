# Design Reference (from KZLMS Modsen project)

Used as visual style reference for Anima app. Not a direct copy — adapted for pet care context.

## Screen size
375 x 812px (iPhone standard)

## Color System — Light Mode

### Primary (Green)
- Main:  #497C60
- Light: #6B957E
- Dark:  #3A634D
- Contrast text: #FFFFFF

### Secondary (Gold/Amber)
- Main:  #D0A85D
- Light: #D9B97D
- Dark:  #A7874B
- Contrast text: #332E37

### Error (Red)
- Main:  #BE4B37
- Light: #E1A69B
- Dark:  #9F2E1A

### Warning (Yellow)
- Main:  #DFA11C
- Light: #E5B449
- Dark:  #B87D00

### Info (Blue-Purple)
- Main:  #5C69A3
- Light: #7D87B5
- Dark:  #4A5482

### Neutrals
- Background:  #F1F4F9
- Surface:     #FFFFFF
- Text dark:   #332E37
- Text darker: #222222
- Muted:       #F6F6F6

## Notes for Anima adaptation
- Keep green as primary (fits nature/animals theme well)
- Replace gold secondary with a warmer tone (coral/peach) - more pet-friendly
- Bottom tabbar pattern: Home | Pets | Calendar | Profile
- Card-based layout for pet list
- Same input field style (rounded corners, floating label)
