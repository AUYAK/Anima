# Localization

## Current state
English only. All screens are built in English until the Localization feature is specced and picked up.

Dev: do not set up i18n infrastructure yet. Hardcoded English strings are fine for now.

## Planned behavior (when Localization feature ships)
- Russian (ru) and English (en) supported
- Language follows device system language: Russian or Belarusian -> Russian, anything else -> English
- No in-app language picker
