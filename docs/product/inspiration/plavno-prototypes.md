# Plavno — Prototypes

Source: https://plavno-deploy.vercel.app/prototype/
Saved: 2026-07-02

---

## Prototype 1 — Редакция (Editorial)
- High-design, minimal functionality
- Full-screen photography + editorial typography
- No user ratings, no social features
- Quiz format: "Лучший ход" (Best Move)
- Philosophy: editorial authority over community

## Prototype 2 — Навигатор (Navigator) v0.3
- Full-featured: voice AI guide ("Спроси Артура"), friend mapping, check-ins, gamification, activity feeds
- Most feature-complete early iteration

## Prototype 3 — Атлас (Atlas)
- Hybrid: editorial design + social functionality
- Introduces Плавно AI assistant
- 41 screens
- Features: check-ins, friend networks, maps, achievements

## Prototype 4 — Маршрут (Route)
- Most developed desktop prototype
- Reward Hub system
- Restaurant news feed
- Animated check-in history on maps
- AI-assisted filtering

## Prototype 5 — Тёмный (Dark) ← most relevant for design reference
- Mobile-first React Native
- **Dark theme**
- **Rounded cards: 20px radius**
- **Purple accents**
- Portrait-oriented event cards
- 58 screens

## Visual Design Language (across prototypes)
- Color palette: dark purple background + purple accents
- Typography: Diatype typeface
- UI patterns: rounded cards, masonry grids, portrait-oriented layouts, swipe mechanics
- Card style: hero image, status badge overlay, minimal chrome

## AI Templates (3 variants tested)
1. Swipe-based (Tinder-like mechanics)
2. Tab menu integration
3. Feed-integrated block interface

All templates: 103-restaurant Moscow database, mood/budget/district filtering.
